## 🛠️ Deployment Steps

1. Created an AWS Account.
2. Created an Amazon S3 Bucket.
3. Uploaded website files.
4. Enabled Static Website Hosting.
5. Configured Bucket Policy for public access.
6. Accessed the website using the S3 Website Endpoint.
